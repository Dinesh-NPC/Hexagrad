# 🌟 Pathora  

**Pathora** is a web and mobile platform developed by **Hexagrad** for **SIH Problem Statement SIH25094**.  
It is designed to guide students in discovering career paths, planning academics, and accessing personalized learning resources.  

---

## ℹ️ About
Pathora acts as a comprehensive career and education assistant.  
- Provides **aptitude and psychometric assessments** to suggest suitable career paths.  
- Recommends **videos and webinars** for skill development.  
- Offers **smart notifications, milestone tracking, and college exploration tools**.  

It helps students plan exams, explore scholarships, track progress, and make informed educational decisions efficiently.  

---

## 🚀 Key Features
- **Career Guidance:** Aptitude & psychometric tests, personalized recommendations  
- **Academic Planning:** Smart notifications, meeting scheduler, milestone tracker  
- **College Exploration:** Compare colleges, scholarship info, map integration  
- **Profile Management:** Login/Sign-up, view/update profiles via Firebase  
- **Cross-Platform:** Available as Flutter app and web application  
- **Notifications:** Reminders for competitive exams and important events  

---

## 🛠️ Tech Stack
- **Frontend:** Flutter (App & Web), Dart  
- **Backend / Database:** Firebase (Auth, Firestore, Storage, Cloud Functions)  
- **Tools & Libraries:** Android Studio, Google Maps API, Firebase Cloud Messaging  
- **Other:** Git & GitHub for version control, Provider/Bloc for state management  

---

## 📖 How It Works
1. Users sign up/login via Firebase authentication.  
2. Complete aptitude or psychometric assessments for career guidance.  
3. Receive personalized learning resources (videos, webinars).  
4. Plan academic journey using notifications, milestones, and college tools.  
5. Track progress and update profiles continuously.
